LEGO EV3 running EV3dev

Begin by checking network connection to Rpi:

This is your menu accessible on the EV3 brick controller. This is the interface running EV3dev distribution of Debian. There is also SSH access which we will cover at the end of this document.

Scroll down and  click on

Wireless and Networks

If you connect any peripheral networks they will appear here. There is a menu option ready for Wi-Fi if you connect a linux compatible dongle.

Scroll to

All Network Connection

and click on it.

Select the Wired network with a USB symbol and click on it

This menu enables the user to disconnect, change from DHCP to static and read MAC address with the ENET command.

Here you have to select IPv4 and click on it.

This screen indicate that we are using static IP and it need to be in the same IP-range as our RPi. In this case we use 10.42.0.3

If this is stated in your screen, then you can skip this step, otherwise continue.

Click on Change

This menu enables 3 default settings, DHCP and a custom IP option.

In this screen choose the

Load Linux defaults.

If you follow the instruction correctly you should see something like this.
