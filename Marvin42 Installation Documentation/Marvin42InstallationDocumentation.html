Raspberry Pi

Prerequisites

Dependencies

  • Update apt cache by running: sudo apt update
  • Use apt to install make: sudo apt install make

Root configs

This part is optional but recommended. The most important file in this repo is etc/dhcpcd.conf for setting up the IP addresses.

Start with flashing Raspbian onto a SD-card (with more than 4GB of storage).

When logged into Raspbian, start a terminal and change directory to / (root).

Clone root configs from https://github.com/albrdev/marvin42-rootconfigs then go into that directory and run sudo make install to automatically copy all files to their respective locations.

Note that locales may need some (re)configuration with sudo raspi-config.

User configs

A similar setup can be done with the user’s home directory. Simply follow similar steps as in the above instructions. Clone https://github.com/albrdev/marvin42-userconfigs then go into that directory and run sudo make install. This will install scripts for making your workflow more convenient.

Software

Chirp Daemon

On Raspberry Pi:

Dependencies

     Follow the instructions at https://developers.chirp.io/docs/getting-started/python

  • sudo apt update
  • sudo apt-get install python3-dev python3-setuptools portaudio19-dev libffi-dev libsndfile1
  • sudo pip3 install chirpsdk

Clone repository from https://github.com/albrdev/marvin42_cd

git clone --recurse-submodules https://github.com/albrdev/marvin42_cd.git

cd into the directory and run sudo make install

A chirp config file is is also required. Visit https://developers.chirp.io/applications, register/login, select a protocol(16khz-mono) from the dropdown menu, then select the Python tab and copy/paste the credentials into ~/.chirprc

Configuration file can be found in /etc/marvin42_cdrc (by default, this shouldn’t need any reconfiguration)

You might want to create a directory for the logs in /var/log with permissions for the user, or simply change log location in the /etc/marvin42_cdrc

Autostart is yet to be implemented

Start daemon:        marvin42_cd start

Stop daemon:        marvin42_cd stop

Restart daemon:        marvin42_cd restart

Transmission Daemon / Collision Guard Daemon

On EV3:

Dependencies

  • Update apt cache by running: sudo apt update
  • Use apt to install make: sudo apt install make

Clone repository from https://github.com/albrdev/marvin42_td

git clone --recurse-submodules https://github.com/albrdev/marvin42_td.git

cd into the directory and run sudo make install

Configuration files can be found at /etc/marvin42_tdrc and /etc/marvin42_cgdrc (in our case this should reflect what user should run the daemon)

You might want to create a directory for the logs in /var/log with permissions for the user, or simply change log location in the /etc/marvin42_tdrc and /etc/marvin42_cgdrc

Autostart is yet to be implemented

Start daemon:        marvin42_td start

marvin42_cgd start

Stop daemon:        marvin42_td stop

                        marvin42_cgd stop

Restart daemon:        marvin42_td restart

                        marvin42_cgd restart

Hardware

Assemble instructions

Raspberry Pi

  1. Power source (5V)
  2. USB, type A (for RPi) to type mini B (for EV3)
  3. USB Sound card/AUX cable(3.5mm) adapter
  4. Microphone (3.5mm AUX)

EV3

  1. USB, type A (for RPi) to type mini B (for EV3)
  2. Infrared distance sensor (RJ-12) (front collision detection sensor) (optional)
  3. Infrared distance sensor (RJ-12) (back collision detection sensor) (optional)
  1. Left (large) motor (RJ-12)
  2. Right (large) motor (RJ-12)
